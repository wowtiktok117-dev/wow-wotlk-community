# Parche de Mecánicas (Cliente estándar / no HD)

## Descripción

Este parche añade mejoras visuales para distintas mecánicas de **World of Warcraft: Wrath of the Lich King 3.3.5a** utilizando los recursos del **cliente estándar (no HD)**.

Está diseñado para funcionar sin modificaciones adicionales y es la opción recomendada para los jugadores que utilizan el cliente **estándar (no HD)**.

---

## Requisitos

* Cliente **World of Warcraft: Wrath of the Lich King 3.3.5a** (estándar / no HD).

---

## Instalación

1. Cierra el juego si está abierto.
2. Copia el archivo de este parche en la carpeta **Data** de tu cliente de World of Warcraft.
3. Inicia el juego.

---

## Compatibilidad

* **Cliente:** World of Warcraft: Wrath of the Lich King 3.3.5a (estándar / no HD)
* **Configuración adicional:** No requerida
* **Uso recomendado:** PvE

---

## Aviso

Este parche ha sido diseñado específicamente para el **cliente estándar (no HD)** y **no requiere realizar ningún cambio en PatchMenu**. Basta con copiar el archivo en la carpeta **Data** para comenzar a utilizarlo.

No se recomienda instalar este parche sobre el **cliente HD**, ya que algunas mejoras visuales están adaptadas específicamente a cada versión del cliente.
