## Criterios de selección

- Solo se utilizan repositorios o páginas oficiales del autor.
- No se emplean forks salvo que sean la versión mantenida para WotLK 3.3.5a.
- Todas las versiones se verifican directamente desde el addon antes de ser distribuidas.
- Cualquier cambio de repositorio oficial deberá actualizarse en este documento.

---

## Mantenimiento

Este documento deberá revisarse siempre que:

- Se actualice un addon.
- Se incorpore un nuevo addon al proyecto.
- Cambie el repositorio oficial de un addon.
- Se detecte un fork oficial más actualizado para World of Warcraft 3.3.5a.
