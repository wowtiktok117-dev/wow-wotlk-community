# Profiles

Esta carpeta contiene los **perfiles individuales** de los addons utilizados en **Wowtiktok WotLK Community**.

Cada subcarpeta incluye los archivos necesarios para importar la configuración de un addon de forma independiente, permitiéndote personalizar tu interfaz sin reemplazar la carpeta `WTF` del juego.

---

## ¿Cuándo utilizar estos perfiles?

Utiliza los perfiles de esta carpeta si deseas:

* Importar la configuración de un addon específico.
* Mantener tu configuración actual.
* Instalar únicamente los perfiles que necesites.

---

## ¿Quieres la configuración completa?

Si prefieres instalar toda la interfaz de una sola vez, utiliza la carpeta **`WTF/`** del repositorio.

La carpeta `WTF` contiene todas las configuraciones ya integradas y listas para copiar directamente en tu cliente de **World of Warcraft: Wrath of the Lich King 3.3.5a**, por lo que no es necesario importar los perfiles de forma individual.

---

> **Importante:** Los addons originales del repositorio nunca se modifican. Todas las personalizaciones se distribuyen mediante perfiles individuales o mediante la carpeta `WTF`, facilitando las actualizaciones y manteniendo una estructura limpia, escalable y fácil de mantener.
