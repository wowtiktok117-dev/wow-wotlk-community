## 📋 Reglas

- El perfil ha sido verificado para **World of Warcraft: Wrath of the Lich King 3.3.5a (Warmane)**.

---

## 📦 Instalación

1. Abre la configuración de **Grid2**.
2. Accede al apartado **Perfiles**.
3. Importa `Grid2_Wowtiktok_Profile.txt`.
4. Ejecuta el siguiente comando para aplicar correctamente todos los cambios:

```text
/reload
```
