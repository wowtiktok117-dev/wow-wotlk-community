## 🎯 Objetivo

Facilitar la importación del perfil oficial de **Deadly Boss Mods (DBM)** recomendado por **Wowtiktok WotLK Community**.

---

## 📂 Contenido

Esta carpeta incluye el siguiente perfil:

- `DBM_Wowtiktok_Profile.txt`

---

## 📋 Reglas

- El perfil ha sido verificado para **World of Warcraft: Wrath of the Lich King 3.3.5a (Warmane)**.
- El perfil ha sido verificado con **Deadly Boss Mods (DBM) 10.1.13_alpha**.
- Se respeta la estructura y el formato originales de DBM.

---

## 📦 Instalación

1. Abre la configuración de **Deadly Boss Mods (DBM)**.
2. Accede al apartado **Profiles**.
3. Importa `DBM_Wowtiktok_Profile.txt`.
4. Ejecuta el siguiente comando para aplicar correctamente todos los cambios:

```text
/reload
```
