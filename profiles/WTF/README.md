# WTF

Esta carpeta contiene la **configuración completa** de la interfaz utilizada en **Wowtiktok WotLK Community**.

Incluye la carpeta **`SavedVariables`**, que contiene los archivos **`.lua`** y **`.lua.bak`** donde se almacena toda la configuración de la interfaz, incluyendo los perfiles de los addons esenciales, los addons recomendados y las WeakAuras, dejándolos listos para usar desde el primer inicio del juego.

---

## ¿Cuándo utilizar esta carpeta?

Utiliza esta carpeta si deseas instalar toda la interfaz de una sola vez.

---

## Instalación

1. Cierra **World of Warcraft** si está abierto.
2. Abre la carpeta donde tienes instalado el juego.
3. Accede a la siguiente ruta:

```text
World of Warcraft 3.3.5a HD (o el nombre de tu cliente)
└── WTF
    └── Account
        └── <Nombre de tu cuenta>
```

4. Copia la carpeta **`SavedVariables`** incluida en este repositorio dentro de la carpeta de tu cuenta **o**, si ya dispones de una carpeta `SavedVariables`, copia su contenido dentro de la carpeta `SavedVariables` de tu cuenta.
5. Si Windows solicita reemplazar los archivos existentes, acepta la operación.

> **Importante:** Antes de reemplazar la carpeta `SavedVariables` o sus archivos, realiza una copia de seguridad de tu configuración actual para poder restaurarla en caso necesario.

---

## ¿Qué contiene?

La carpeta **`SavedVariables`** incluida en este repositorio almacena:

* La configuración completa de los addons esenciales.
* La configuración completa de los addons recomendados.
* Todas las WeakAuras integradas y listas para usar.

---

## ¿Cuál es la diferencia con la carpeta `profiles`?

La carpeta **`profiles`** contiene los perfiles de importación de cada addon por separado.

La carpeta **`WTF`** ya incorpora todas esas configuraciones integradas y preparadas para funcionar desde el primer inicio del juego.

---

> **Importante:** Los archivos incluidos en esta carpeta están diseñados para proporcionar una configuración completa y lista para usar. Si ya utilizas una interfaz personalizada, se recomienda realizar una copia de seguridad de tu carpeta `SavedVariables` antes de reemplazarla.
