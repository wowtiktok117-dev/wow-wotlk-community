# WTF

Esta carpeta contiene la **configuración oficial** de **Wowtiktok WotLK Community** para **World of Warcraft: Wrath of the Lich King 3.3.5a (Warmane)**.

Permite instalar de una sola vez todos los perfiles oficiales compatibles incluidos en el proyecto.

---

## 🎯 Objetivo

Facilitar una instalación completa de la interfaz recomendada por la comunidad, ofreciendo una configuración homogénea, verificada y lista para utilizar.

---

## 📂 Contenido

La carpeta `WTF` incluye la configuración oficial de los recursos distribuidos por **Wowtiktok WotLK Community**, manteniendo una experiencia homogénea y verificada.

Actualmente podrá incluir configuraciones para recursos como:

- ElvUI
- Skada
- Method Raid Tools
- WeakAuras (cuando corresponda)

La lista de recursos podrá ampliarse conforme el proyecto incorpore nuevas configuraciones oficiales.

---

## 📋 Reglas

- Solo se incluirán configuraciones verificadas por el proyecto.
- Todas las configuraciones deberán indicar la versión de **World of Warcraft** y la versión de los addons con las que han sido verificadas.
- Antes de copiar esta carpeta se recomienda realizar una copia de seguridad de la carpeta `WTF` existente.
- La configuración distribuida representará siempre la versión oficial recomendada por **Wowtiktok WotLK Community**.

---

## 📦 Instalación

1. Cierra World of Warcraft.
2. Realiza una copia de seguridad de tu carpeta `WTF` actual.
3. Sustituye la carpeta `WTF` del juego por la incluida en este directorio.
4. Inicia el juego.
