# WeakAuras

Esta carpeta contiene las WeakAuras oficiales distribuidas por **Wowtiktok WotLK Community**.

Las WeakAuras permiten ampliar la interfaz del juego mediante avisos visuales, sonoros y personalizados para mejorar la experiencia PvE.

---

## 🎯 Objetivo

Centralizar las WeakAuras oficiales del proyecto para facilitar su instalación, mantenimiento y actualización, garantizando una experiencia homogénea y verificada para **World of Warcraft: Wrath of the Lich King 3.3.5a (Warmane)**.

---

## 📂 Contenido

Esta carpeta podrá incluir:

- WeakAuras para clases.
- WeakAuras para especializaciones.
- WeakAuras para bandas.
- WeakAuras para encuentros específicos.
- Paquetes completos de WeakAuras.
- Recursos complementarios cuando sean necesarios.

Cada colección dispondrá de su propia carpeta cuando sea necesario.

---

## 📋 Reglas

- Solo se incluirán WeakAuras verificadas por el proyecto.
- Todas las WeakAuras deberán indicar la versión de **World of Warcraft** y la versión del addon **WeakAuras** con las que han sido verificadas.
- Siempre que sea posible, cada colección incluirá su propia documentación.
- Las WeakAuras deberán mantener una organización clara y coherente.
- Se priorizarán configuraciones orientadas exclusivamente al contenido PvE.
